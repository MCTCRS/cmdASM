CMD ASM Cheat Sheet

Headers:
  .head SB_OBJ <name>         # runtime scoreboard objective
  .head SB_PREFIX <prefix>    # optional scoreboard prefix
  .head SB_SUFFIX <suffix>    # optional scoreboard suffix
  .head STORAGE <namespace>   # required storage namespace
  .head STORAGE_PATH <path>   # storage path
  .head ENTITY <uuid|RANDOM>  # required entity
  .head EVAL_ENTITY <uuid>    # entity for .eval

Labels:
  @label                      # define label

Flow:
  .jump <label> [condition]   # jump to label
  .call <label>               # call label (use .ret to return)
  .ret                        # return from .call
  .sleep <ticks>              # wait ticks
  .sleep (command)            # wait ticks returned from command
  .eval '<json>'              # flatten JSON text into command

Notes:
  - Lines not starting with '.' are raw Minecraft commands
  - Leading whitespace is allowed
